# Archflow

Archflow helps teams design infrastructure visually while keeping the model in
Git as YAML. It bridges the gap between architecture and implementation by
turning intent into validated, structured models that can drive NetBox as the
operational backend and produce inventory and IaC-oriented outputs for tools
such as Ansible and Terraform.

It is built for logical, network, and physical architecture design, with the
UI acting as a projection over the model rather than the source of truth.

## Getting Started

- Read `docs/architecture.md` for the stable architecture overview.
- Review `docs/decisions.md` for current design decisions.
- Inspect `examples/minimal.yaml` for a minimal model shape.
